# Mi Boleta Transparente

Proyecto web para la gestión y visualización transparente de boletas.

## Características
- Gestión de boletas
- Sistema de asistencia
- Procesamiento de datos
- Resultados en tiempo real

## Requisitos
- PHP 7.4 o superior
- MySQL 5.7 o superior
- Servidor web (Apache, Nginx)
- Laragon (para desarrollo local)

## Instalación
1. Clonar el repositorio: `git clone https://github.com/tu-usuario/mi_boleta_transparente.git`
2. Importar la base de datos (si aplica)
3. Configurar conexión a base de datos
4. Acceder a `http://localhost/mi_boleta_transparente`

## Estructura del Proyecto
- `/api` - Endpoints de API
- `/assets` - Imágenes, iconos, fuentes
- `/css` - Hojas de estilo CSS
- `/data` - Archivos de datos y configuración
- `/includes` - Archivos PHP incluidos
- `/js` - Archivos JavaScript
- `/logs` - Registros del sistema
- `/asistencia` - Módulo de asistencia
- `/mantenimiento` - Herramientas de mantenimiento
- `/procesar` - Scripts de procesamiento
- `/resultados` - Páginas de resultados
